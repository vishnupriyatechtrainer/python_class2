## ðŸ”¹ **Method 2: Upload files using Git & Terminal (Recommended for developers)**



* Git must be installed â†’ [Download Git](https://git-scm.com/)
* GitHub account



1. **Create a repository on GitHub** (as in Method 1)

2. **Open your terminal or command prompt**

3. **Navigate to your project folder:**

   ```bash
   cd path/to/your/folder
   ```

4. **Initialize git:**

   ```bash
   git init
   ```

5. **Add your GitHub repo as remote:**

   ```bash
   git remote add origin https://github.com/your-username/your-repo-name.git
   ```

6. **Add all files to staging:**

   ```bash
   git add .
   ```

7. **Commit the files:**

   ```bash
   git commit -m "Initial commit"
   ```

8. **Push files to GitHub:**

   ```bash
   git branch -M main
   git push -u origin main
   ```
